# タイピングゲーム

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1
* 必要なものがあれば追記してください（非推奨）

## ゲームの概要
* 主人公キャラクターZENをタイピングにより敵を倒していくゲームで，．．．
* 参考URL：[サイトタイトル](https://www.hoge.com/)

## ゲームの遊び方
* 敵が主人公にあったったら、，ゲームオーバーとなる

## ゲームの実装
* タイピング
### 共通基本機能
* 背景画像と主人公キャラクターの描画

### 分担追加機能

* 敵のスピードダウン（担当：小澤）：使用回数が決まっているスピードダウンの機能の実装
* 敵の種類追加(担当：小出)：敵がいっぱいでる機能の実装
* アルティメット機能:(担当:高本)あるボタンをクリックすると、どんな文字を打っても敵に攻撃できるようになる機能
* 音声追加(担当：金子)：操作をするたびに効果音がなる機能
* ゾンビタイム(担当：菊池)：一定時間敵が倒せなくなる。敵をその場にとどめることしかできない。
* 画像をいれる（担当：北條）：敵の速度を１定にし、時間経過で速度を上げるようにする。

### メモ
* 文字がおかしかったり、打てない文字がある。
